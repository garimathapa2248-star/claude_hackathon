# Debt Decoder

Debt Decoder helps NJIT students upload financial aid documents and understand their real cost, loan risks, scholarship conditions, and next steps in plain English.

## Features
- Financial aid document upload
- Term decoder
- Red flag scanner
- AI financial aid assistant
- School comparison
- Appeal letter generator
- Emergency aid resources

## Tech Stack
- HTML
- CSS
- JavaScript
- Claude/OpenAI API planned
- PDF/text extraction planned

## How to Run
Open index.html in your browser.
